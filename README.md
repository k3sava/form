# FORM — Poster Maker

Type a phrase, pick a feeling, export a poster. Nine real-time effects render the text as a moving field — breath, murmuration, smoke, tide, grain, mycelium, bloom, condensation, phantom. Square, portrait, landscape. PNG still or 30-second video loop.

Runs entirely in the browser. No accounts, no uploads, nothing leaves the page.

Live at **[toys.iamkesava.com/poster-maker](https://toys.iamkesava.com/poster-maker/)**.

## What's in here

- `index.html` — the whole app. One file, vanilla JS, custom Simplex noise + Boids, Canvas 2D.
- `generate_demos.py` — Pillow-based renderer that replays a few of the effects offline for thumbnails / printing.

## Themes

Dark by default, light theme toggle in the header. Preference persists per device.
