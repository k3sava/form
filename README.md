# FORM

Type a phrase, the system arranges it like a typographer. A small browser toy and a quiet typography lab in one file.

Live at **[toys.iamkesava.com/form](https://toys.iamkesava.com/form/)**.

Runs entirely in the browser. No accounts, no uploads, nothing leaves the page.

## What's in here

- `index.html` — the whole app. One file, vanilla JS, Canvas 2D.
- `generate_demos.py` — Pillow-based renderer that reproduces a few of the effects offline for thumbnails / printing.

## Themes

Dark by default, light theme toggle in the header. Preference persists per device.
